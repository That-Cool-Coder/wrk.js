wrk.attitude.copySub()

Reference contents

Syntax:

wrk.attitude.copySub(a1, a2)

Example:

var attitude3 = wrk.attitude.copySub(attitude, attitude2);

Description:

Return a new attitude that is the difference between ```a1``` and ```a2```. Doesn't edit ```a1``` or ```a2```.
