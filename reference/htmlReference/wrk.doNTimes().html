wrk.doNTimes()

Reference contents

Syntax:

wrk.doNTimes(n, func)

Example:

wrk.doNTimes(5, (i) => { newRecords[i] = records[i] });

Description:

Run ```func``` n times. Pass the current loop count (from 0 to ```n - 1```) as an argument to func. Shorthand for ```for (var i = 0; i < n; i ++) func(i);```.
