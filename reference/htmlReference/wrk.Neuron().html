wrk.Neuron()

Reference contents

Syntax:

wrk.Neuron(bias=wrk.randflt(-1, 1))

Example:

var neuron = new wrk.Neuron(0.2)

Description:

A generic neuron class. Designed to primarily be used by wrk.neuralNetwork. ```bias``` shouldn't ever need to be set.
