wrk.randint()

Reference contents

Syntax:

wrk.randint(min, max)

Example:

var result = wrk.randflt(0, 4)

Description:

Returns a random integer in the half-open range ```[min, max)```. This includes ```min``` but not ```max```. Possible results from the example would be [0, 1, 2, 3].
