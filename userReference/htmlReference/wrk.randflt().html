wrk.randflt()

Reference contents

Syntax:

wrk.randflt(min, max)

Example:

var result = wrk.randflt(0, 1)

Description:

Returns a random float in the half-open range [min, max). This includes min but not max. Possible results in the example would range from 0.0 to 0.999...
