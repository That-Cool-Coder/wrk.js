wrk.dom.logPara

Reference contents

Syntax:

wrk.dom.logPara

Example:

var elem = wrk.dom.logPara

Description:

A paragraph element which can be used to log data to. Only created when wrk.dom.logToPara is called. Has access methods wrk.dom.clearLogPara and wrk.dom.logToPara
